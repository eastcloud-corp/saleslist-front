## 技術スタック

- **フロントエンド**: Next.js 14 (App Router)
- **スタイリング**: Tailwind CSS + shadcn/ui
- **認証**: JWT Bearer Token
- **状態管理**: React Query (TanStack Query)
- **API**: Django DRF (将来移行予定)
